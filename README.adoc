== 使い方

=== 小説を読む

[source,bash]
----
./bin/soundnovel start <novel_id>
----

=== 小説のIDを調べる

[source,bash]
----
./bin/soundnovel list
----

== 小説の追加

`lib/soundnovel/*.novel` にテキストファイルを配置する。
書式は他のファイルを読めばだいたい分かると思う。
